# PsRoom Enterprise - Sistem Informasi Reservasi Room Rental PlayStation

Proyek ini adalah sistem informasi berbasis web yang dirancang untuk mengelola operasional bisnis rental PlayStation secara digital dan terintegrasi. Dibangun sebagai Proyek Akhir untuk mata kuliah **Pemrograman Web (Semester 2)** di STT-NF.

## 🚀 Fitur Utama
- **Sistem Reservasi Real-Time:** Proses booking room rental berdasarkan durasi jam yang fleksibel.
- **Manajemen Data Master (Filament Admin):** Pengelolaan data pelanggan, room, kategori PlayStation, hingga manajemen user (Admin/Staf).
- **Penghitungan Otomatis:** Sistem menghitung total biaya sewa secara otomatis berdasarkan durasi main.
- **Laporan Pemesanan (View):** Halaman rekap dan visualisasi laporan transaksi untuk mempermudah pemilik bisnis memantau pendapatan.

## 🛠️ Tech Stack
- **Framework:** Laravel 11
- **Admin Panel:** Filament v3
- **Database:** MySQL
- **Frontend/Styling:** TailwindCSS, Blade Components
